DYNAMIC port forwarding

Allows to proxy traffic through compromised server.
Creates SOCKS4 proxy.


Example

Tunneling out the web traffic

-C enable compression
-T do not open interactive shell on remote host
-N don't start shell command

plink.exe -ssh remote-username@tunnel-to-ssh-server-host.com -pw PASSWORD123 -C -T -D 127.0.0.1:10022 -N
