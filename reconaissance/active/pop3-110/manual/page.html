Connect to the pop3 server

telnet mail.yourdomain.com 110

Submit your username to the server

user <user>

Submit your password to the server

pass <pass>

List the emails in the inbox, including their ID number, and size

list

Read email

If you want to read an email from the list, type retr # where # is the number of the email you want read.
For example, to read the 2nd messge in the list, retr 2

retr <mail #>

Delete email

This will delete an email. After you issue the dele command, you put a number.
This is the number of the email you want to delete.
So for example, to delete message 1, type dele 1

dele <mail #>

Save changes and quit

quit
