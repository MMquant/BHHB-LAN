Switches

-sS SYN scan
-sT Full TCP 3-hand shake scan
-p ports
--top-ports <how many> scans top <how many> ports
-oG <file> output to greppaable file
-sn ping sweep
-v verbose
-O operating system fingerprinting
-sV service enumeration/banner grabbing
-A More aggressive scan, more information about host, more traffic generated
-iL input filename with hosts to scan
-p- scan all ports 1-65535

Scripts

--script smb-os-discovery.nse SMB service OS discovery
--script-help smb-os-discovery.nse prints script help

NSE scripts path

/usr/share/nmap/scripts
