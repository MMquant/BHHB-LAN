Open the session to the IMAP server

telnet mail.yourdomain.com 143

Send the username and password to the system (the "a" is a required part of the command)

a LOGIN <user> <pass>

Shows a listing of all the IMAP folders the account has

a LIST "" "*"

Select the INBOX folder. You can repalce INBOX with any folder you wish to use

a SELECT INBOX

List all the emails in the selected folder, and will also tell you the flags like "seen" or "NonJunk"

a fetch 1:* flags

List the header information of message 1 in the selected folder

a fetch 1 body[header]

List the header AND the contents of message 1 in the selected folder

a fetch 1 body[]

Set message 1 to be deleted

It will not delete until the expunge command has been run

a store 1 +flags deleted

Delete any emails that you flagged to be deleted with the previous command

a expunge

Disconnect from the IMAP server

a logout
